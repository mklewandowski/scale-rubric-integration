#  SCALE Science Learning-Centered Rubric integration
The SCALE Science Learning-Centered Rubric site is a single page application (SPA) built using TypeScript and React. The site contains multiple rubrics and allows users to choose a rubric, enter student data, and obtain performance information. The site does not connect to a database or a backend server and user entered data is not transmitted to another site or location. All user data is stored in browser local storage which is stored only on the user’s computer. Entered data can be exported/imported as a JSON file, and performance summaries can be exported as CSV, XLSX, or PDF files. Entered data can be deleted by the user.

The content in this repository is designed to help integrate the SCALE Science Learning-Centered Rubric site into the C2C platform. The goal is to access C2C teacher, student, and performance data and provide a simplfied version of the SCALE Science Learning-Centered Rubric site which mainly provides summary and assessment information.
